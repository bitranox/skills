# CSS Types

CSS types define the values that Textual CSS styles accept. They are referenced from the [styles reference](../styles/index.md) in the "Formal Syntax" section of each style, denoted by a keyword enclosed in angle brackets.

## Contents

| Page                          | Description                                                    |
|-------------------------------|----------------------------------------------------------------|
| [<border>](border.md)         | Represents a border style.                                     |
| [<color>](color.md)           | Represents a color.                                            |
| [<hatch>](hatch.md)           | Represents a character used in the hatch rule.                 |
| [<horizontal>](horizontal.md) | Represents a position along the horizontal axis.               |
| [<integer>](integer.md)       | Represents an integer number.                                  |
| [<keyline>](keyline.md)       | Represents a line style used in the keyline rule.              |
| [<name>](name.md)             | Represents a sequence of characters that identifies something. |
| [<number>](number.md)         | Represents a real number (integer or decimal).                 |
| [<overflow>](overflow.md)     | Represents overflow modes.                                     |
| [<percentage>](percentage.md) | Represents a percentage value.                                 |
| [<pointer>](pointer.md)       | Represents pointer (cursor) shapes for the mouse.              |
| [<position>](position.md)     | Defines how the offset rule is applied.                        |
| [<scalar>](scalar.md)         | Represents a length (number with a unit, or auto).             |
| [<text-align>](text_align.md) | Represents alignments that can be applied to text.             |
| [<text-style>](text_style.md) | Represents styles that can be applied to text.                 |
| [<vertical>](vertical.md)     | Represents a position along the vertical axis.                 |
