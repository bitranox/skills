# Links

Styles that influence the appearance of inline action links within a widget.

These styles only target Textual action links, not internet hyperlinks.

## Contents

| Page | Description |
|------|-------------|
| [Link-background](link_background.md) | Sets the background color of the link. |
| [Link-background-hover](link_background_hover.md) | Sets the background color of the link when the cursor is over it. |
| [Link-color](link_color.md) | Sets the color of the link text. |
| [Link-color-hover](link_color_hover.md) | Sets the color of the link text when the cursor is over it. |
| [Link-style](link_style.md) | Sets the text style for the link text. |
| [Link-style-hover](link_style_hover.md) | Sets the text style for the link text when the cursor is over it. |
