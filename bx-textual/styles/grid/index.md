# Grid

Styles relating to the Textual `grid` layout.

For an in-depth look at the grid layout, visit the grid [guide](../../guide/layout.md#grid).

## Contents

| Page | Description |
|------|-------------|

| [Grid-columns](grid_columns.md) | Defines the width of the columns of the grid. |
| [Grid-gutter](grid_gutter.md) | Sets the spacing between adjacent cells in the grid. |
| [Grid-rows](grid_rows.md) | Defines the height of the rows of the grid. |
| [Grid-size](grid_size.md) | Sets the number of columns and rows in a grid layout. |
| [Row-span](row_span.md) | Specifies how many rows a widget will span in a grid layout. |
