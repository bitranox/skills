# Scrollbar Colors

Styles to set the colors used in Textual scrollbars.

## Contents

| Page | Description |
|------|-------------|
| [Scrollbar-background](scrollbar_background.md) | Sets the background color of the scrollbar. |
| [Scrollbar-background-active](scrollbar_background_active.md) | Sets the scrollbar background color when the thumb is being dragged. |
| [Scrollbar-background-hover](scrollbar_background_hover.md) | Sets the scrollbar background color when the cursor is over it. |
| [Scrollbar-color](scrollbar_color.md) | Sets the color of the scrollbar thumb. |
| [Scrollbar-color-active](scrollbar_color_active.md) | Sets the scrollbar thumb color when it is being dragged. |
| [Scrollbar-color-hover](scrollbar_color_hover.md) | Sets the scrollbar thumb color when the cursor is over it. |
| [Scrollbar-corner-color](scrollbar_corner_color.md) | Sets the color of the gap between horizontal and vertical scrollbars. |

## See also

 - [`scrollbar-size`](../scrollbar_size.md) to set the dimensions of scrollbars.
 - [`scrollbar-gutter`](../scrollbar_gutter.md) to reserve space for a vertical scrollbar.
